What is Cloudinary?

Cloudinary is a cloud-based media storage and management service.
Instead of storing images on your own server, you upload them to Cloudinary, and it stores them for you.
Think of it as Google Drive, but specifically designed for images and videos in web applications.




Can MongoDB store images?

Yes. MongoDB can store images.
There are two common ways:
    1.Store the image as Binary (Buffer/BinData) inside a document.
    2.Store large files using GridFS, which splits files into chunks.
So the reason we don't usually store images in MongoDB is not because it's impossible. It's because it's not the best tool for the job.
